# Comparador Justo de Protocolos

## Principio fundamental

**Rara vez hay un "mejor absoluto".** La mayoria de debates en fitness (PPL vs UL, keto vs balanced, HIIT vs LISS) son falsas dicotomias. La respuesta casi siempre es "depende del contexto".

---

## Framework de 6 dimensiones

Cada protocolo se evalua en 6 dimensiones con una escala de 1-5:

### 1. Evidencia de eficacia
- ¿Hay estudios comparando DIRECTAMENTE ambos protocolos?
- ¿Que tipo de estudios? (meta-analisis > RCT > observacional)
- ¿Cual es el effect size? (trivial < 0.2 | pequeno 0.2-0.5 | moderado 0.5-0.8 | grande > 0.8)
- Si no hay comparacion directa: ¿hay evidencia mecanistica que sugiera superioridad?

### 2. Aplicabilidad a la poblacion
- ¿Funciona para principiantes? ¿Intermedios? ¿Avanzados?
- ¿Funciona para hombres y mujeres?
- ¿Funciona para jovenes y mayores?
- ¿Hay poblaciones para las que esta contraindicado?

### 3. Sostenibilidad (>12 semanas)
- ¿Se puede mantener a largo plazo?
- ¿Genera fatiga excesiva? ¿Relacion insana con la comida/ejercicio?
- ¿Requiere restricciones sociales extremas?
- ¿Es compatible con una vida normal (trabajo, familia, social)?

### 4. Adherencia esperada
- ¿Es realista para humanos imperfectos?
- ¿Requiere mediciones imposiblemente precisas?
- ¿Es flexible ante imprevistos (viajes, enfermedad, estres)?
- **La adherencia es el predictor #1 de resultados.** El mejor plan es el que puedes seguir.

### 5. Riesgos y efectos secundarios
- ¿Hay riesgo de lesion? ¿De deficiencia nutricional?
- ¿Hay riesgo psicologico (relacion insana, TCA)?
- ¿Hay contraindicaciones medicas?
- ¿Cual es el peor escenario realista?

### 6. Contexto del usuario
- Fase actual (deficit, mantenimiento, superavit)
- Dias disponibles para entrenar
- Equipamiento disponible
- Experiencia y preferencias
- Condiciones medicas

## Matriz de comparacion estandarizada

```
COMPARACION: [Protocolo A] vs [Protocolo B]
============================================

Resultado medido: [Fuerza / Hipertrofia / VO2max / Composicion corporal]
Nivel de evidencia: [Meta-analisis / RCTs / Observacional / Opinion]

| Dimension                | Protocolo A (1-5) | Protocolo B (1-5) | Notas |
|--------------------------|-------------------|-------------------|-------|
| 1. Evidencia de eficacia | /5                | /5                |       |
| 2. Aplicabilidad         | /5                | /5                |       |
| 3. Sostenibilidad        | /5                | /5                |       |
| 4. Adherencia            | /5                | /5                |       |
| 5. Riesgos               | /5 (5=bajo riesgo)| /5                |       |
| 6. Contexto usuario      | /5                | /5                |       |
| TOTAL                    | /30               | /30               |       |

VEREDICTO: [A superior / B superior / Equivalentes / Depende de X]
PARA QUIEN ES MEJOR A: [perfil de persona]
PARA QUIEN ES MEJOR B: [perfil de persona]
```

## Comparaciones pre-analizadas

### Splits de entrenamiento

#### PPL x2 (6 dias) vs Upper/Lower (4 dias)

| Dimension | PPL x2 | Upper/Lower | Notas |
|-----------|--------|-------------|-------|
| Evidencia | 4/5 | 4/5 | Ambos con frecuencia 2x/semana, hipertrofia similar |
| Aplicabilidad | 3/5 | 5/5 | PPL requiere 6 dias, UL solo 4 |
| Sostenibilidad | 3/5 | 5/5 | PPL genera mas fatiga acumulada |
| Adherencia | 3/5 | 4/5 | Mas facil comprometerse a 4 dias |
| Riesgos | 3/5 | 4/5 | PPL puede sobrecargar hombros (push/pull frecuente) |
| **Veredicto** | Mejor si tienes 6 dias y buena recuperacion | Mejor para la mayoria de personas con vida ocupada |

#### Full Body (3 dias) vs Upper/Lower (4 dias)

| Dimension | Full Body x3 | Upper/Lower x2 | Notas |
|-----------|-------------|-----------------|-------|
| Evidencia | 4/5 | 4/5 | Frecuencia 3x vs 2x: meta diferencia minima |
| Aplicabilidad | 5/5 | 4/5 | FB con solo 3 dias, ideal principiantes |
| Sostenibilidad | 5/5 | 4/5 | Menos dias = mas sostenible |
| Adherencia | 5/5 | 4/5 | Menos compromiso semanal |
| Riesgos | 5/5 | 4/5 | Menor volumen por sesion, menos fatiga |
| **Veredicto** | Mejor para principiantes y personas con <4 dias | Mejor para intermedios con 4 dias disponibles |

---

### Protocolos de nutricion

#### Dieta keto vs Dieta balanceada (en deficit)

| Dimension | Keto | Balanceada | Notas |
|-----------|------|------------|-------|
| Evidencia | 3/5 | 5/5 | Meta-analisis: sin diferencia con calorias igualadas (Hall 2017) |
| Aplicabilidad | 2/5 | 5/5 | Keto no apta para todos (restriccion severa) |
| Sostenibilidad | 2/5 | 5/5 | Keto dificil de mantener a largo plazo |
| Adherencia | 2/5 | 4/5 | Muchos abandonan keto por restriccion social |
| Riesgos | 3/5 | 5/5 | Keto puede comprometer rendimiento en fuerza |
| **Veredicto** | Solo si el usuario la prefiere y adhiere | Superior para la mayoria de personas |

#### Ayuno intermitente 16:8 vs Comidas distribuidas

| Dimension | IF 16:8 | Distribuidas (4-5) | Notas |
|-----------|---------|---------------------|-------|
| Evidencia | 3/5 | 4/5 | Distribucion de proteina ligeramente mejor para sintesis muscular |
| Aplicabilidad | 4/5 | 4/5 | Ambas adaptables |
| Sostenibilidad | 4/5 | 4/5 | IF mas simple, distribuidas mas flexibles |
| Adherencia | 4/5 | 4/5 | Depende de la persona |
| Riesgos | 3/5 | 4/5 | IF riesgo de atracones, dificil alcanzar proteina en ventana corta |
| **Veredicto** | Herramienta de adherencia, no superior per se. NO recomendado con historial de TCA |

### Protocolos de cardio

#### HIIT vs LISS (para perdida de grasa)

| Dimension | HIIT | LISS | Notas |
|-----------|------|------|-------|
| Evidencia | 4/5 | 4/5 | Gasto calorico similar → resultados similares en composicion corporal |
| Aplicabilidad | 3/5 | 5/5 | HIIT demandante, no apto para todos |
| Sostenibilidad | 3/5 | 5/5 | HIIT causa mas fatiga, limita a 1-2x/semana |
| Adherencia | 3/5 | 4/5 | LISS mas facil de mantener |
| Riesgos | 2/5 | 5/5 | HIIT mayor riesgo lesion, interfiere con piernas |
| **Veredicto** | 80% LISS + 20% HIIT es la combinacion optima |

## Verificar honestidad de una comparacion

Cuando alguien afirma "X es mejor que Y", verificar:

| Trampa | Pregunta clave |
|--------|---------------|
| Mejor caso de A vs peor caso de B | ¿Se estan comparando en igualdad de condiciones? |
| Variables no controladas | ¿Calorias, proteina, volumen son iguales? |
| Efecto novedad | ¿El resultado es porque es NUEVO, no porque es mejor? |
| Sesgo del autor | ¿Vende alguno de los dos protocolos? |
| Generalizacion | ¿El estudio es en la misma poblacion que el usuario? |
| Tiempo insuficiente | ¿<4 semanas? Los resultados iniciales de cualquier cambio son engañosos |
| Outcome selectivo | ¿Se mide lo que importa o un proxy? |

---

## Regla de oro para comparaciones

> **Si ambos protocolos tienen evidencia razonable y el effect size es similar, la decision se reduce a: ¿cual puede SEGUIR el usuario de forma consistente durante meses?**

La adherencia supera la optimizacion teorica en el 99% de los casos.
