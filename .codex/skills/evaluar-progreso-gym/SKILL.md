# Evaluador de Progreso en el Gimnasio

## Indicadores a revisar

### 1. Fuerza en ejercicios clave

Los mejores indicadores de retencion/ganancia muscular:

| Ejercicio | Tipo | Que indica |
|-----------|------|------------|
| Sentadilla | Compuesto inferior | Cuadriceps, gluteos |
| Peso muerto / RDL | Compuesto inferior | Cadena posterior |
| Press banca | Compuesto superior | Pecho, triceps |
| Press hombros | Compuesto superior | Deltoides, triceps |
| Dominadas/Jalon | Compuesto superior | Dorsal, biceps |
| Remo | Compuesto superior | Espalda media |

**Interpretacion:**
- Fuerza se mantiene o sube → **retienes/ganas musculo**
- Fuerza baja ligeramente (5-10%) → Normal en deficit prolongado, monitorear
- Fuerza cae significativamente (>10%) → **Posible perdida muscular, ajustar**

### 2. Peso corporal (promedio semanal)

| Velocidad de perdida | Interpretacion |
|----------------------|----------------|
| 0.5-1% peso/semana | Optimo para recomposicion |
| 1-1.5% peso/semana | Algo agresivo, vigilar fuerza |
| >1.5% peso/semana | Demasiado rapido, riesgo de perdida muscular |
| Sin cambio 2-3 semanas | Posible estancamiento, ajustar calorias |
| Ganancia de peso | Revisar deficit calorico |

### 3. Mediciones corporales

Medir semanalmente en las mismas condiciones:
- **Cintura** (a la altura del ombligo): Debe bajar si pierdes grasa
- **Caderas**: Puede bajar
- **Brazos** (relajados y flexionados): Deben mantenerse o subir
- **Muslos**: Deben mantenerse
- **Pecho**: Debe mantenerse

**Buen indicador**: Cintura baja + brazos/muslos se mantienen = pierdes grasa, retienes musculo

### 4. Fotos de progreso (cada 4 semanas)

- Frente, lado y espalda
- Misma iluminacion, camara, angulo, hora, pose
- Comparar mes a mes (NO semana a semana — cambios muy sutiles)

### 5. Composicion corporal

- **DEXA scan**: Gold standard (cada 2-3 meses)
- **Bioimpedancia**: Menos precisa pero util para tendencias (mismas condiciones siempre)

---

## Senales de alarma

### Estas perdiendo musculo si:
- Fuerza cae >10% en 2+ ejercicios compuestos en 2-3 semanas
- Pierdes >1.5% peso/semana de forma sostenida
- Brazos y muslos se achican pero cintura no baja proporcionalmente
- Fatiga persistente que no mejora con descanso
- Perdida de definicion muscular, zonas "blandas"

### Acciones correctivas:
1. **Reducir deficit** (subir 100-200 kcal)
2. **Asegurar proteina** (minimo 1.8 g/kg, mejor 2.0-2.4 g/kg)
3. **Revisar sueno** (7-9 horas)
4. **Considerar diet break** (1-2 semanas a mantenimiento)
5. **Reducir cardio** si es excesivo
6. **Hacer deload** si no has hecho en 4+ semanas


## Cuando ajustar el plan

| Situacion | Accion |
|-----------|--------|
| Fuerza sube + peso baja | Perfecto, no cambiar nada |
| Fuerza mantiene + peso baja 0.5-1%/sem | Optimo, continuar |
| Peso no baja 2-3 semanas | Reducir 100-150 kcal o anadir 1 sesion LISS |
| Fuerza baja significativamente | Subir calorias 100-200 kcal, asegurar proteina y sueno |
| Peso baja >1.5%/semana | Subir calorias 200-300 kcal |
| Fatiga cronica | Deload inmediato + revisar sueno/estres |
