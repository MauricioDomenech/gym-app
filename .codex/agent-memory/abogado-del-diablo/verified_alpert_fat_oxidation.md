## Alpert Fat Oxidation Rate — Verified March 2026

### Original study:
- Alpert (2005) "A limit on the energy transfer rate from the human fat store in hypophagia"
- Published value: 290 +/- 25 kJ/kg/day = ~31.5 kcal/lb/day
- Based on data from Minnesota Starvation Experiment
- PubMed: https://pubmed.ncbi.nlm.nih.gov/15615615/

### CRITICAL CORRECTION:
- Alpert subsequently identified a miscalculation
- Corrected value: approximately **22 kcal/lb/day** (~48 kcal/kg fat/day)
- This correction was NOT republished in the paper before his passing
- The corrected value is LESS commonly cited in fitness circles

### Practical impact:
For a person with 21.4 kg (47.2 lbs) of body fat:
- Using WRONG value (31 kcal/lb): max deficit = ~1464 kcal/day
- Using CORRECT value (22 kcal/lb): max deficit = ~1038 kcal/day
- Difference of ~426 kcal/day — VERY significant

### How to apply:
ALWAYS use the corrected 22 kcal/lb value when calculating maximum fat loss rate. The 31 kcal/lb figure that circulates widely in fitness media is the UNCORRECTED version. This matters most for lean individuals but affects everyone's calculations.

### Nuckols rule relationship:
The Nuckols rule (BF% / 20 = % weight loss/week) is a practical heuristic that roughly aligns with the corrected Alpert value for moderate body fat percentages. It should be treated as a MAXIMUM tolerable rate, not a recommended target.
